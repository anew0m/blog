관계형 데이터 모델링 노트 개정판 - 김기창저

관계형 데이터 모델링 노트 개정판은 위즈덤마인드의 김기창 대표님이 집필한 책이다.

본 도서는 기존에 발간된 관계형 데이터 모델링 노트 - 재미와 깨달음이 있는 모델링 이야기 프리미엄 가이드 2를 개선되어 재발간된 책으로, 2018.03.31에 출간되었으며 총 742 페이지로 아래와 같은 구성으로 이루어져있다.

  1. 엔터티 이야기

  2. 정규화 이야기

  3. 데이터 통합과 서브타입 이야기

  4. 속성 이야기

  5. 관계 이야기

  6. 이력 데이터 이야기

  7. 비정규화 이야기

본 도서를 요약 및 정리하고자 생각하게된 계기는 네이버 카페인 데이터와사람들에서 진행된 스터디를 시작하게됨으로써 도서를 요약 및 정리하기로 결심했다.

따라서 책 내용을 정리하여 포스팅 하는 주기는 스터디 일정에 맞춰 19.10.05일 부터 2주 간격으로 한 장씩 포스팅 하고자하지만 유동적으로 변경될 수도 있을것 같다.
